# Scripts_Master_Thesis
Here they are, the scripts presented and used in the Master's Thesis "Modeling Injector Effects in Hybrid Rocket Engines Numerical Simulations"
In the Master's Thesis, by Alessio De Luca at Sapienza University of Rome, there are specific explanations on how to use the codes present in this repository.
As in the Thesis Appendix, they are split into UDF and Utility Scripts.
